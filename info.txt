### **Prompt / Instruksi Proyek: Website Generator Surat Cuti Tahunan**

Buat sebuah **website generator surat permohonan cuti tahunan** dengan fitur sebagai berikut:

1. **Fitur Utama** (BISA DI GANTI)

   * Input data personal: **nama, jabatan, alamat**
   * Input detail cuti: **tanggal mulai, tanggal akhir, alamat selama cuti**
   * Input informasi surat: **nomor surat, tanggal pembuatan**
   * Preview surat otomatis di sisi kanan layar
   * Tombol aksi: **Buat Surat**, **Hapus**, dan **Download PDF**

2. **Teknologi yang Digunakan**

   * **Frontend:** React + Vite
   * **Styling:** CSS eksternal, warna dominan **kuning dengan gradasi lembut**
   * **Backend (opsional):** Node.js

3. **Langkah Instalasi Awal Proyek Vite**

```bash
npm create vite@latest generator-surat-permohonan -- --template react
cd generator-surat-permohonan
npm install
npm run dev
```

4. **Struktur Kode Modular dan Maintainable**

   * Pisahkan file `.jsx` dan `.css` ke folder yang sesuai
   * Gunakan folder **components/**, **styles/**, dan **pages/**
   * Setiap komponen (form input, preview surat, tombol aksi, header, footer) dibuat **terpisah**
   * Gunakan **CSS eksternal**, hindari styling inline

5. **Desain UI**

   * Warna dominan: **kuning dengan gradasi lembut**
   * Layout responsif: **form input di kiri**, **preview surat di kanan**
   * Tombol “Buat Surat” dan “Hapus” jelas, interaktif, dan mudah dijangkau
   * Header menampilkan **logo Nuansa Lega** (gunakan URL image)
   * Footer sederhana untuk informasi tambahan

6. **Fungsi Tambahan**

   * Tombol **Download PDF** untuk mengekspor hasil surat menjadi dokumen PDF
   * Preview surat diperbarui secara otomatis saat input berubah
   * Tombol **Hapus** untuk membersihkan semua input

7. **Catatan Teknis**

   * Gunakan React state untuk mengelola input form
   * Gunakan `ref` dan library seperti `html2canvas` + `jsPDF` untuk download PDF
   * Pastikan kode modular agar mudah dikembangkan atau diubah di masa depan
